# asw-000-ciao-mondo

Il progetto **asw-000-ciao-mondo** contiene una sola applicazione **ciao-mondo**, 
che non è un'applicazione distribuita. 

## Build  

Per la compilazione e l'assemblaggio, vedere le istruzioni 
descritta nella sezione [projects/](../). 

## Ambiente di esecuzione 

L'applicazione **ciao-mondo** può essere eseguita nell'ambiente **standalone**, 
sul nodo **node**. 

## Esecuzione 

Per eseguire l'applicazione **ciao-mondo**: 

1. posizionarsi nella cartella principale dell'applicazione 

2. usare il comando `gradle run`

In alternativa: 

1. posizionarsi nella cartella **dist** dell'applicazione 

2. eseguire lo script `../run-ciao-mondo.sh`
